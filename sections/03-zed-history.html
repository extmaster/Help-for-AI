3. Навигация по истории чатов в Zed

Важно: Чаты хранятся локально на компьютере. Между разными компьютерами история не синхронизируется автоматически.

Быстрый доступ к недавним чатам

Действие Mac Linux/Windows
6 последних чатов Cmd+Shift+J Ctrl+Shift+J
Вся история чатов Cmd+Shift+H Ctrl+Shift+H

Также можно кликнуть на иконку меню (...) в правом верхнем углу Agent Panel и выбрать "View All".

Использование старого чата как контекст

В новом чате набери @ и выбери нужный thread из списка — его содержимое добавится как контекст.

Сохранение чата для другого компьютера

Экспорт в Markdown

Action: agent: open thread as markdown

Сохрани файл в облако или Git репозиторий

AGENTS.md

Ключевые договорённости держи в файле AGENTS.md в репозитории

Он будет на всех машинах при клонировании

Новый чат из текущего с резюме

Когда контекст переполняется, Zed предложит создать новый чат с summary текущего. Можно сделать это вручную:

  • Меню Agent Panel → "New from Summary"
