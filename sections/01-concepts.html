1. Ключевые концепции

MCP (Model Context Protocol) Стандарт

Что это: Открытый стандарт (как USB-C для ИИ) для подключения AI-приложений к внешним системам — базам данных, инструментам, API.

Зачем нужен:

  • Подключить ИИ к Sentry, GitHub, Figma, поисковикам документации
  • Расширить возможности без написания кода
  • Стандартизировать интеграции
Rules / AGENTS.md Инструкции

Что это: Файлы с инструкциями для ИИ о твоём проекте.

Зачем нужны:

  • ИИ понимает структуру проекта
  • Следует твоим code standards
  • Не нужно повторять объяснения
Skills (Навыки агента) Переиспользуемые

Что это: Переиспользуемые инструкции для специфических задач (релизы, код-ревью и т.д.)

Навыки загружаются по запросу — агенты видят доступные навыки и могут загрузить полное содержимое при необходимости.
