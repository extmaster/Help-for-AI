6. Эффективная работа с ИИ

Почему тратится много времени и денег без результата:

Проблема Решение
ИИ не понимает проект Создай AGENTS.md с описанием структуры
Повторяю одно и то же Добавь инструкции в rules
ИИ делает не то Используй Plan mode сначала, потом Build
Много токенов уходит Ограничь MCP серверы, не подключай всё подряд
Контекст теряется между компьютерами Храни важное в AGENTS.md в Git

Формула хорошего запроса

[Контекст] + [Конкретная задача] + [Ограничения] + [Формат вывода]

Плохой запрос

«Сделай авторизацию»

Хороший запрос

«Добавь JWT авторизацию в @src/api/auth.ts. Используй jsonwebtoken. Токен храни в httpOnly cookie. Покажи план перед реализацией.»

Основные команды OpenCode

Команда Описание
/init Инициализация проекта, создание AGENTS.md
/undo Откатить последние изменения
/redo Вернуть откаченные изменения
/share Поделиться ссылкой на разговор
Tab Переключение между Plan/Build режимами
@filename Упомянуть файл в промпте

Действие Mac Linux/Windows
